## Action Needed

- **Playwright 1.57 — Chromium → Chrome for Testing** (recent 2026 release)
  - Playwright now uses Chrome for Testing builds instead of Chromium for both headed and headless mode.
  - This changes the rendering engine used for screenshots. UI review route crawler baselines may produce different pixel output.
  - **Action:** After upgrading Playwright, re-capture reference screens and re-baseline visual comparisons. Expect minor rendering differences.
  - Affects: `project-audit.md` → Visual Verification, UI Review System (route crawler, capture sets, reference screens)

- **Playwright 1.58 — Removed deprecated selectors** (recent 2026 release)
  - Removed `_react` and `_vue` selectors entirely. Use locators guide alternatives.
  - Removed `:light` selector engine suffix. Use standard CSS selectors.
  - Removed `Devtools` option from `BrowserType.LaunchAsync()`.
  - Removed macOS 13 (Ventura) support for WebKit.
  - **Action:** Before upgrading, audit any Playwright scripts in projects for usage of `_react`, `_vue`, or `:light` selectors. Fix before upgrade.
  - Affects: `project-audit.md` → Visual Verification, UI Review route crawler

  - Affects: All long-running CC sessions, scheduled tasks, extended development sessions.

  - Affects: Anyone editing global CLAUDE.md or cross-project files with conditional skills active.

- **CC v2.1.86 — Marketplace plugin permissions fix** (March 27)
